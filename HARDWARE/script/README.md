### OpenTSN3.4-HARDWARE-script

#### 概述

script介绍的是OpenTSN3.4的硬件实例化工程，包含网卡示例工程和交换机示例工程。
#### 结构
script
    ├── TSNNic3.4_FPGA_2port：     //实例化2个接口（一个主机接口和一个网络接口）的TSN网卡示例工程，使用的quartus版本为Quartus Prime Standard Edition 19.1，使用的FPGA型号为Intel Arria10:10AX048H2F34E2SG.
    └── TSNSwitch3.4_FPGA_4port    //实例化4个接口的TSN交换机示例工程，使用的quartus版本为Quartus Prime Standard Edition 19.1，使用的FPGA型号为Intel Arria10:10AX048H2F34E2SG.
